# Ink Diffusion 结论记录

## 当前结论（已完成）

- 最小链路已跑通：
  - `高斯初值 -> initial.csv -> PDE simulation -> trajectory.csv -> supervised -> surrogate_predictions.csv + surrogate_eval_report.csv`
- 数值验证通过：`stable_cfl=True`，`mass_delta_abs≈0`，`status=PASS`。
- 监督学习验证通过：在当前一阶局部扩散任务上，回归误差接近 0。

## 意义判断

- 作为研究最终题目：墨水扩散一步预测本身偏基础，创新度有限。
- 作为方法论验证：价值很高，已经证明管线正确、可复现、可扩展。

## 关于“不同墨水/不同液体”的可行性

可以，但要明确边界条件：

- 在当前模型里，核心控制量是扩散系数 `kappa`（或等价扩散参数）。
- “不同墨水 in 不同液体”可以先抽象为不同 `kappa` 的任务族。
- 若要更真实，需要逐步加入：
  - 浓度相关扩散（非线性）
  - 对流项（液体流动）
  - 温度/黏度影响
  - 反应项（如果有化学反应）

结论：

- **能做预测**，但当前版本是“简化物理场景”的预测，不是完整流体-化学系统。

## 下一步建议（用于升级研究价值）

1. 做参数反演：由观测反推 `kappa`。  
2. 做部分观测预测：稀疏测点重建全场。  
3. 做多步 rollout 评估：验证长期稳定性和失效边界。
